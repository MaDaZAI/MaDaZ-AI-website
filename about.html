    Project Overview

    MaDaZ-AI System Manager is a customizable multi-agent AI platform designed for any user and domain without requiring programming skills.

    It includes specialized agents such as SystemManagerAgent, ModelTunerAgent, and other user-defined agents , built on a scalable container-based architecture and deployment options on cloud servers or locally.

    Technologies Used

    Python · Docker · WebSocket · SSH (WIP) · LLMs · CLI
